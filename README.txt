设计性实验：图书借阅系统

小组成员：陈光霁、马得宇、田正阳、钱志浩、朱超。

功能说明：
1、该系统主程序在“图书馆管理系统.cpp”中，文件中有五个头文件分别为：c1.h(文件包含和宏定义）、
define.h（程序链表中各类数据类型定义）、xfile.h(支持文件存取功能函数）、function.h（链表各接口
）、application.h（各模块功能实现函数）。

2、该系统能实现以下功能：
学生建档：建立学生档案，录入学生信息；
学生销档：删除学生档案中学生信息（同时消除借阅卡信息）；
借阅卡发卡：向已存在的学生发放借阅卡，产生唯一对应的卡号；
借阅卡销户：删除某学生借阅卡信息（不影响学生档案信息）；
图书入库：添加图书信息（若该图书已存在，则进行数量累加）；
借书：持有借阅卡的学生可进行借书操作（借期两个月），并添加借阅记录；
还书：对已存在借阅记录的学生进行还书操作（改变图书库存量），并删除该项记录；
借阅情况查询：查询所有的借阅记录；
现存图书查询：查询所有的图书信息；
退出系统：完成各项操作后，把新信息存入文件中；
注：该系统有一定的容错功能，由于学生水平有限，程序中仍存在Bug，未能尽善尽美。
